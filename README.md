? Kendra Browser
?Peramban Web Python GUI Cepat dan Ringan untuk Lingkungan Linux (Dioptimalkan untuk Termux VNC)
?Kendra Browser adalah proyek open source yang dibangun dengan Python dan PyQt5, memanfaatkan QtWebEngine untuk rendering cepat. Dirancang untuk berjalan di lingkungan GUI yang ringan seperti Termux VNC, browser ini menawarkan fitur dasar modern termasuk tab, download manager, dan Developer Tools (Inspect Element).
?

(Catatan: Harap ganti USER/REPO dengan nama pengguna GitHub dan nama repositori Anda setelah Anda mengunggahnya!)
?? Fitur Utama
?Multiple Tabs: Dukungan penjelajahan dengan banyak tab.
?Download Manager: Penanganan permintaan unduhan dengan dialog simpan.
?URL Bar Cerdas: Navigasi cepat dengan address bar yang responsif.
?Developer Tools: Akses alat inspeksi web (Inspect Element).
?Optimasi Termux: Memanfaatkan perpustakaan Qt bawaan Termux.
